Projeto Votorantim 2.0
Engenharia Química - Viabilidade técnica e econômica

Esse projeto foi consequência de um projeto de gestão de resíduos prévio da Poli Junior, em que uma solução dada para o projeto foi a produção de zeólitas. O cliente ficou tão satisfeito com o projeto que decidiu estudar a possibilidade de criação de uma planta industrial de zeólitas no Brasil e, para isso, contratou a Poli Junior. Nesse projeto, estudamos os custos de instalação e produção, as possibilidades de vendas das zeólitas para empresas próximas ao possível local de instalação, etc.
