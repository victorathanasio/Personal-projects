Projeto Zuqueiro
Engenharia Química - Gestão de Resíduos

A Zuquero é uma empresa que oferece soluções ambientais e tecnologias inovadoras para colocar essas soluções em prática. Eles procuraram a Poli Jr porque desejavam encontrar uma funcionalidade para um resíduo proveniente de um processo de pirólise. Esse projeto deu para o cliente diversas possibilidades do que fazer com esse resíduo, para que fosse reaproveitado de maneira ecológica.
